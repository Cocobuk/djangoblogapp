# ✍️ YB Blog - Django Blog Uygulaması

Bu proje, geliştiricilerin makalelerini paylaşabileceği, yorum alabileceği ve içeriklerini yönetebileceği tam kapsamlı bir **Django** blog platformudur. Dinamik yapısı ve kullanıcı yönetim sistemiyle modern bir web deneyimi sunar.

## ✨ Özellikler

- **Kullanıcı Sistemi:** Kayıt olma, giriş yapma ve güvenli çıkış işlemleri.
- **Makale Yönetimi:** Makale oluşturma, düzenleme, silme ve detaylı görüntüleme.
- **Kontrol Paneli:** Kullanıcıların kendi yazdığı makaleleri takip edebileceği özel dashboard.
- **Yorum Sistemi:** Makalelere dinamik olarak yorum ekleyebilme özelliği.
- **Arama Fonksiyonu:** Anahtar kelimeye göre makaleler içinde arama yapma.
- **CKEditor Entegrasyonu:** Makale yazarken zengin metin düzenleme (Kod blokları, görseller vb.).
